# ⚓ Modern Warships AI Battle Analyzer

AI-powered gameplay analyzer for the Modern Warships community.  
Upload a screenshot or video clip — get instant AI coaching on your battle performance.

## Features
- 🎬 Auto frame extraction from gameplay videos (MP4, MOV)
- 🤖 AI analysis powered by Claude (Anthropic)
- 📡 Covers: Positioning · Weapon usage · Minimap awareness · HP management
- 🇰🇷 Korean-language coaching output
- 🎯 Severity rating + 3 actionable improvement tips per battle

## Purpose
Non-commercial community project built by a 4-year Modern Warships player  
(consistently ranked top 50 globally) currently studying AI/ML at university.
